## Overview
A production-grade full-stack web application built on AWS for a real-world
bus body building company. This project demonstrates hands-on experience in
cloud architecture, serverless backend development, authentication, and
secure API design.

## Architecture
- Frontend hosted on Amazon S3 and delivered via CloudFront
- Backend APIs using Amazon API Gateway and AWS Lambda
- Authentication with Amazon Cognito (JWT based)
- Data storage using Amazon DynamoDB
- Monitoring and logging via Amazon CloudWatch
- Secure access using IAM least-privilege policies

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: AWS Lambda (Node.js)
- Database: DynamoDB
- Authentication: Amazon Cognito
- Infrastructure: AWS (Serverless)
- Monitoring: CloudWatch
